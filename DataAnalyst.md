# Python for Data Analysts: Syllabus

This track focuses on using Python for data manipulation, analysis, and visualization.

## 1. Prerequisites Refresh
- Lists, Dictionaries, and List Comprehensions
- Working with CSV files basics

## 2. Numerical Computing with NumPy
- N-dimensional arrays (ndarrays)
- Vectorized operations and Broadcasting
- Slicing and Indexing arrays
- Linear Algebra basics and Random number generation

## 3. Data Manipulation with Pandas
- Series and DataFrames
- Loading data from various formats (CSV, Excel, SQL)
- Data Cleaning: Handling missing values, duplicates, and type conversion
- GroupBy operations and Pivot Tables
- Merging, Joining, and Concatenating datasets

## 4. Data Visualization
- **Matplotlib**: Basic plotting (Line, Bar, Scatter, Histogram)
- **Seaborn**: Advanced statistical visualizations (Heatmaps, Violin plots, Pair plots)
- Customizing charts (Labels, Titles, Legends, Themes)

## 5. Statistical Analysis
- Descriptive Statistics (Mean, Median, Std Dev, Correlations)
- Introduction to Hypothesis Testing with `scipy.stats`
- Exploratory Data Analysis (EDA) workflows

## 6. Capstone Project
- Analyzing a real-world dataset (e.g., Titanic, Netflix, or Housing data) and presenting insights.
