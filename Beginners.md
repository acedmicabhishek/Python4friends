# Python for Beginners: Syllabus

Welcome to the Python journey! This track is designed to take you from zero to being able to write your own scripts and solve problems with code.

## 1. Getting Started
- Installation and Setup (Python, VS Code)
- The Python REPL vs. Scripting
- Comments and Code Documentation

## 2. Fundamentals
- **Variables & Data Types**: Strings, Integers, Floats, Booleans
- **Basic Operators**: Arithmetic, Comparison, Logical
- **Input & Output**: using `print()` and `input()`

## 3. Control Flow
- **Conditional Statements**: `if`, `elif`, `else`
- **Loops**: `for` loops (iterating over ranges and lists) and `while` loops
- **Break & Continue**: Controlling loop execution

## 4. Data Structures
- **Lists**: Creating, indexing, slicing, and methods (`append`, `pop`, etc.)
- **Dictionaries**: Key-value pairs and their use cases
- **Tuples & Sets**: Understanding immutability and uniqueness

## 5. Functions & Modules
- Defining and calling functions
- Arguments and Return values
- Scope (Local vs. Global)
- Importing standard library modules (e.g., `math`, `random`)

## 6. Project: Small CLI App
- Building a simple calculator or a text-based adventure game.
